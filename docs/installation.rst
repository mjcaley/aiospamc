############
Installation
############

************
Requirements
************

* Python 3.8 or later
* SpamAssassin running as a service

*******
Install
*******

With pip
========

.. code-block:: bash

    pip install aiospamc

With git
========

.. code-block:: bash

    git clone https://github.com/mjcaley/aiospamc.git
    cd aiospamc
    poetry install

.. note::
    aiospamc's build system uses Poetry which you can get from here: https://python-poetry.org/
