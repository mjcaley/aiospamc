############
Installation
############

************
Requirements
************

* Python 3.9 or later
* SpamAssassin running as a service

*******
Install
*******

With pip
========

.. code-block:: bash

    pip install aiospamc

With git
========

.. code-block:: bash

    git clone https://github.com/mjcaley/aiospamc.git
    cd aiospamc
    uv sync

.. note::
    aiospamc's build system uses uv which you can get from here: https://docs.astral.sh/uv/
